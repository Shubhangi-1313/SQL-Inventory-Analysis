# SQL Inventory Analysis

## Overview
This is a beginner SQL project created using an inventory dataset. The project demonstrates basic SQL queries to analyze inventory records.

## Skills Used
- CREATE TABLE
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)

## Tools
- SQL Server
- SQL Server Management Studio (SSMS)

## Files
- `Inventory_Database.sql`
- `Inventory-Records-Sample-Data.xlsx`

## Outcome
This project helped me practice SQL fundamentals and improve my data analysis skills.
